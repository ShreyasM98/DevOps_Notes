Linux Commands
Linux OS acts as interface between your hardware of your system and software on the machine

To clear the screen
$clear

To see the working directory
$pwd(present working directory);

To check the commands executed in the current session
$history

To create a directory 
$mkdir fileName

To create a file
$touch fileName

To check the directory 
$ls

To check about complete information about a command
$man mkdir -- gives complete details about the command

To check about linux commands
$man mkdir -- tells all about the mkdir command

To create recursive directories - One folder inside another folder
$ sudo mkdir -p dir1/dir2/dir3

To come out of the directory
$ cd ..

echo - To print or insert to a file
$ echo "Learning Linux" > file1.txt

cat - To show the content in the file
$ cat file1.txt

vim - to open the editor in the linux
$ vim file2.txt

head - to open the first 5 lines of the text
$ head -n 5 file2.txt

tail - to open the last 5 lines of the text file
$ tail -n 5 file2.txt

sed - to find the particular line in a file
$ sed -n '5,10p' file2.txt

grep - to find a particular word in a file
$ grep "toolchains" file2.txt

cp - copy from existing file to new file
$ cp file1.txt newcopiedfile

mv - Move the file to the directory
$ sudo mv newcopiedfile dir1

mv - Rename the file with new file
$ mv dummy.txt renamefile.txt

rm - remove file in the directory
$ rm file1.txt

rm -rf  --> to remove the directory
$ sudo rm -rf dir
